# Algorithm

알고리즘 문제 풀이

## 개발 환경
- Java 17
- IDE: IntelliJ IDEA

## 풀이 방법

### 1. 노트에 풀이 정리
- 문제 이해 및 요구사항 분석
- 입력/출력 예시 확인
- 제약 조건 파악
- 알고리즘 접근 방법 구상
- 시간 복잡도 / 공간 복잡도 예상

### 2. 코드 정리
- 구상한 알고리즘을 코드로 구현
- 가독성 있는 코드 작성
- 필요시 주석 추가

### 3. 성능 확인
- 테스트 케이스 실행
- 시간 복잡도 검증
- 공간 복잡도 검증
- 최적화 가능 여부 검토

## 커밋 예시
#### 예시
```bash
문제명

- 시간복잡도: O(n log n)
- 공간복잡도: O(1)
- 풀이방법: 배열 정렬 후 양 끝에서 투포인터로 탐색
```
